# Bank Churn Analysis using Excel

## Project Overview
This project analyzes customer churn behavior in a banking dataset using Microsoft Excel Pivot Tables and statistical analysis techniques.

## Objectives
- Analyze customer churn patterns
- Compare customer balances and credit scores
- Study churn rates based on banking products
- Perform statistical research analysis

## Data Cleaning
- Replaced missing values in EstimatedSalary using average values
- Replaced inconsistent Balance value using average balance

## Tools Used
- Microsoft Excel
- Pivot Tables
- Statistical Analysis
- Data Cleaning Techniques

## Analysis Performed
1. Average credit score by geography
2. Average balance by gender and geography
3. Active vs inactive customers by credit card
4. Churn rate by number of products
5. Credit score comparison across tenure and churn status
6. Research question analysis using statistical interpretation

## Key Findings
- Spain showed the highest average credit score
- German female customers had the highest balances
- Customers with credit cards were slightly more active
- No significant difference found in salary between churned and non-churned customers
- No significant difference found in credit scores among geographic locations
